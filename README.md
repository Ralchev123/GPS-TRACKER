# 🚀 GPS-T

---

## 📦 Инсталация

1. 👉 Създай `.env` файл в root директорията на проекта.
2. 👉 Инсталирай нужните зависимости с:

    ```
    npm install
    ```

---

## 🏁 Стартиране на сървъра
.
    ```
    node server.js
    ```

✅ Сървърът вече трябва да работи!  
🌐 Отвори браузъра си и провери на: `http://localhost:3000`.
